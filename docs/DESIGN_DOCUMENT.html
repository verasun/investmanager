1 项目概述

项目简介

InvestManager 是一个功能完整的金融量化分析系统，支持A股和美股市场数据获取、技术分析、策略回测、风险管理以及自动化报告生成。系统采用前后端分离架构，提供RESTful API服务和交互式Web仪表板。

📈

技术分析

30+技术指标与形态识别

🔄

策略回测

完整回测引擎与绩效评估

🛡️

风险管理

仓位管理与风险预警

核心功能

数据采集模块

  • 美股数据：Yahoo Finance API
  • A股数据：Akshare 开源库
  • 数据清洗与标准化处理
  • 特征工程与技术指标计算

分析引擎

  • 技术指标：MA、EMA、RSI、MACD、KDJ等
  • 形态识别：19种蜡烛图形态
  • 信号生成：买入/卖出/持有信号
  • 趋势分析与支撑阻力位

回测系统

  • Backtrader风格策略框架
  • 多策略对比分析
  • 完整绩效指标计算
  • 交易记录与组合价值追踪

风险控制

  • 多种仓位管理算法
  • 风险敞口监控
  • 压力测试
  • 实时预警通知

    def generate_signals(self, data: pd.DataFrame) -> pd.Series:

5.4.2 绩效指标 (PerformanceMetrics)

指标 公式 说明
总收益率 (期末值 - 期初值) / 期初值 策略整体收益
年化收益 (1 + 总收益率)^(252/n) - 1 年化后的收益率
夏普比率 (年化收益 - 无风险利率) / 年化波动率 风险调整后收益
索提诺比率 (年化收益 - 无风险利率) / 下行波动率 下行风险调整收益
最大回撤 max((峰值 - 当前值) / 峰值) 最大亏损幅度
胜率 盈利交易数 / 总交易数 交易胜出比例
盈亏比 平均盈利 / 平均亏损 单笔盈亏对比
盈利因子 总盈利 / 总亏损 整体盈亏比

算法 说明 公式
FixedSizer 固定金额 size = fixed_amount / price

7.6 常见问题

Q: 数据获取失败怎么办？

A: 检查网络连接，确认API限制。YFinance有请求频率限制，建议添加缓存或使用代理。

Q: 回测结果与预期不符？

A: 检查策略参数设置，确认数据完整性，注意交易成本和滑点的影响。

Q: 如何添加自定义策略？

A: 继承BaseStrategy类，实现generate_signals方法，然后在策略模块中注册。
